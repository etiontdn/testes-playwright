Elementos Textuais do HTML e Exemplos Práticos

Introdução aos Elementos Textuais

Se necessário use os exemplos à baixo para entender como usar esses elementos:

Elementos Semânticos

Elementos semânticos são aqueles que transmitem significado sobre o seu conteúdo tanto para o navegador quanto para o desenvolvedor. Eles ajudam a criar uma estrutura lógica para a página, melhorando a compreensão por parte de motores de busca e tecnologias assistivas.

<main>

O elemento <main> representa o conteúdo dominante do <body>. Ele deve conter o conteúdo único de um documento, ou seja, o conteúdo que não é repetido em outras páginas (como barras laterais, links de navegação, informações de direitos autorais repetidas, logotipos de sites e formulários de pesquisa, exceto se o formulário de pesquisa for a função principal do documento).

<section>

<h{n}> (<h1> a <h6>)

Os elementos <h1> a <h6> representam os seis níveis de cabeçalhos de seção de um documento. <h1> é o mais importante (geralmente o título principal da página), e <h6> é o menos importante. Use-os de forma hierárquica para estruturar seu conteúdo.

Elementos Livres de Significado (Não Semânticos)

Estes elementos são usados principalmente para fins de estilo ou agrupamento de conteúdo quando não há um elemento semântico mais apropriado. Eles não carregam nenhum significado inerente.

<span>

O elemento <span> é um contêiner em linha genérico para fraseado de conteúdo, que não representa semanticamente nada. É usado para aplicar estilos a uma parte de um texto ou para manipular uma parte de um texto via JavaScript.

Este é um texto com uma parte destacada com span para estilo.

<div>

O elemento <div> é um contêiner de bloco genérico para conteúdo. Não representa semanticamente nada e é usado quando nenhum outro elemento semântico é apropriado para agrupar elementos para fins de estilo (com CSS) ou manipulação (com JavaScript).

Outros Elementos Textuais Importantes

Além dos elementos principais, HTML oferece diversos outros elementos para refinar o significado e a apresentação do texto.

<strong> e <b>

O elemento <strong> indica que seu conteúdo tem forte importância, seriedade ou urgência. O elemento <b> (bold) é usado para chamar a atenção para o texto sem transmitir importância adicional.

Este texto é importante (strong).
Este texto é apenas em negrito (b).

<em> e <i>

O elemento <em> (emphasis) é usado para dar ênfase a uma palavra ou frase, o que geralmente é renderizado em itálico. O elemento <i> (italic) é usado para texto que deve ser apresentado em itálico por uma razão diferente de ênfase, como termos técnicos, expressões idiomáticas ou pensamentos.

Eu *realmente* amo HTML (ênfase).
O termo biológico para cães é Canis familiaris (itálico).

<mark>

O elemento <mark> representa texto que é destacado ou marcado para fins de referência ou anotação.
