Termos e Definições

HTML

Linguagem de Marcação de Hipertexto, usada para criar a estrutura de páginas web.

CSS

Folhas de Estilo em Cascata, usadas para estilizar o conteúdo HTML.

JavaScript

Linguagem de programação que permite adicionar interatividade e dinamismo às páginas web.
