Exemplos de Listas em HTML

As listas HTML são usadas para agrupar conjuntos de itens relacionados de forma organizada. Existem vários tipos para diferentes propósitos:

Listas Não Ordenadas (`<ul>`)

São usadas para agrupar itens que não têm uma ordem específica. Geralmente são renderizadas com marcadores (bolinhas).

<ul>
    <li>Item um</li>
    <li>Item dois</li>
    <li>Item três</li>
</ul>

Exemplo Visível:

  • Item um
  • Item dois
  • Item três

Listas Ordenadas (`<ol>`)

Usadas para agrupar itens que precisam ser numerados ou ordenados (ex: passos de uma receita, rankings). São renderizadas com números ou letras.

<ol>
    <li>Primeiro passo</li>
    <li>Segundo passo</li>
    <li>Terceiro passo</li>
</ol>

Exemplo Visível:

  1. Primeiro passo
  2. Segundo passo
  3. Terceiro passo

Listas Aninhadas

É possível aninhar listas uma dentro da outra para criar estruturas hierárquicas, combinando `<ul>` e `<ol>` conforme a necessidade.

<ul>
    <li>Frutas
        <ul>
            <li>Maçã</li>
            <li>Banana</li>
        </ul>
    </li>
    <li>Vegetais
        <ol>
            <li>Cenoura</li>
            <li>Alface</li>
        </ol>
    </li>
</ul>

Exemplo Visível:

  • Frutas
    • Maçã
    • Banana
  • Vegetais
    1. Cenoura
    2. Alface

Listas de Definição (`<dl>`)

São usadas para criar uma lista de termos e suas respectivas definições, como em um glossário.

<dl>
    <dt>HTML</dt>
    <dd>Linguagem de Marcação de Hipertexto.</dd>
    <dt>CSS</dt>
    <dd>Folhas de Estilo em Cascata.</dd>
</dl>

Exemplo Visível:

HTML
Linguagem de Marcação de Hipertexto.
CSS
Folhas de Estilo em Cascata.
