O Elemento <link> no HTML

O elemento <link> é fundamental em páginas HTML para estabelecer relações do documento atual com recursos externos. Ele é sempre utilizado dentro da seção <head> do HTML e não aparece visivelmente na página, ao contrário de outros elementos do corpo.

Sua principal função é indicar quais folhas de estilo (CSS) podem ser usadas na página, permitindo que o design e a estilização sejam aplicados. Por exemplo, para vincular uma folha de estilo externa, usamos: 

<link rel="stylesheet" href="estilo.css">

Os dois atributos mais importantes do elemento <link> são:

  • href: Define a URI (Uniform Resource Identifier) do recurso alvo, ou seja, o caminho para o arquivo externo.
  • rel: Define a relação do recurso externo com o documento atual, especificando o tipo de relação (ex: "stylesheet" para folhas de estilo).

Em resumo, o <link> atua como uma ponte entre o seu documento HTML e outros recursos importantes para sua renderização e funcionalidade, por exemplo, nesta página usamos o Materialize CSS para estilização simplesmente adicionando um link.
