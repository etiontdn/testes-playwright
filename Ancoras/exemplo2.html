O Elemento <a> (Âncora) no HTML

O elemento <a>, também conhecido como "âncora", é utilizado para definir um hiperlink. Ele permite que você conecte o seu documento atual a um recurso externo, seja outra página web, um arquivo para download, ou até mesmo uma seção específica dentro da mesma página.

Diferentemente de elementos como <link>, o elemento <a> só pode aparecer dentro do <body> do documento HTML, pois ele é parte do conteúdo visível e interativo da página.

O atributo mais importante para o elemento <a> é o href. Ele define a URI (Uniform Resource Identifier) do recurso alvo, ou seja, para onde o link irá apontar. Por exemplo, para criar um link para o Google, você faria: 

<a href="http://google.com">Google</a>
Google

Além do href, existem outros atributos úteis que podem ser usados para controlar o comportamento do link:

  • rel: Define a relação do recurso linkado com o documento atual (ex: "nofollow", "noopener", "noreferrer").
  • download: Quando presente, indica que o recurso deve ser baixado diretamente pelo navegador quando o link é clicado, em vez de ser navegado.
  • target: Especifica onde o recurso deve ser aberto. Valores comuns incluem _blank (abre em uma nova aba ou janela), _self (abre na mesma janela/aba - padrão), _parent, ou _top.
